# TripSitter

TripSitter is a collaborative travel planning app that lets you build and manage trip itineraries, visualise your travel history and future, and share trips with friends for a seamless group planning experience.

---

## Features

- **Itinerary Management** — Create and organise day-by-day plans for your trips with locations of places to visit and your stays
- **Interactive Map** — Visualise past and upcoming destinations on a live map
- **Upcoming Trips** — See everything on your horizon at a glance
- **Collaborative Planning** — Share trips with others and plan together in real time
- **Achievements** - Gain badges for your travels

---

## Status

TripSitter is currently in **early development**. Features are actively being built and some functionality may be incomplete or subject to change.

---

## Tech Stack

- **Frontend** — [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool** — [Vite](https://vitejs.dev/)

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/tripsitter.git
   cd tripsitter
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`
